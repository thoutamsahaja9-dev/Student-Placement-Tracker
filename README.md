Student Placement Tracker

A full-stack web application to manage students, companies, and placement records. The application provides a centralized platform for tracking placement activities and visualizing key statistics through an interactive dashboard.

Features

- Student management (Create, Read, Update, Delete)
- Company management (Create, Read, Update, Delete)
- Placement management (Create, Read, Update, Delete)
- Dashboard with placement statistics
- Student search functionality
- RESTful API integration
- Responsive user interface

Tech Stack

Frontend

- React
- JavaScript
- HTML
- CSS

Backend

- FastAPI
- SQLAlchemy

Database

- PostgreSQL

Deployment

- Vercel (Frontend)
- Render (Backend)

Project Structure

Student-Placement-Tracker/
├── frontend/
└── backend/

Getting Started

1. Clone the repository.
2. Install frontend and backend dependencies.
3. Configure the PostgreSQL database.
4. Start the FastAPI backend.
5. Start the React frontend.
